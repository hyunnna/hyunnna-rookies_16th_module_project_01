# < Rookies 16th Module Project 1 >


### 프로젝트 소개
* * *

AWS 서비스, 오픈 소스 보안 라이브러리를 사용해 가용성과 보안성을 보장하는 게시판 웹 애플리케이션을
제작 및 클라우드 환경에서 배포하고, 자체 취약점을 진단 후 보완하였습니다.

</br>

|:----:|:---:|
|웹 애플리케이션 제작|2023.12.26 ~ 2023.12.30|

|웹 애플리케이션 배포|2023.12.30|
